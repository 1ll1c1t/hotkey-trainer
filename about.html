About This Project

Hotkey Trainer is an interactive educational tool designed to help creatives master the complex keyboard shortcuts of professional design and animation software.

Learning software like Blender, Maya, or the Adobe Suite can be overwhelming. Menus are deep, and clicking buttons breaks your creative flow. The fastest way to work is by using the keyboard, but memorizing hundreds of keys is difficult.

This site solves that problem by turning dry lists of shortcuts into an explorable, interactive dashboard. You can:

  • Search for any action (e.g., "Extrude", "Render") to find the key instantly.
  • Visualize how shortcuts are categorized (Navigation vs. Tools).
  • Test your muscle memory with the built-in Quiz Mode.

This project is open source and hosted on GitHub. Feel free to explore the tools and speed up your workflow today.
